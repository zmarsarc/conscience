# API Descriptions

## /contracts/
    GET: return all contrasts.
    POST: add new contract.
    
    /contracts/<id>/
        GET: return the contract which specify by id.
        PUT: alert the contract.

## /purchases/
    GET: return all orders.
    POST: add new orders.
    
    /purchases/<id>/
        GET: return the order which specify by id.
        PUT: alert the order.
